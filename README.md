# 🛒 Lista de Compras Inteligente em Python

Este é um projeto prático desenvolvido para consolidar meus estudos iniciais em Python. O programa simula uma lista de compras interativa rodando direto no terminal, permitindo gerenciar itens do dia a dia.

## 🚀 Funcionalidades
- **Adicionar Itens:** Permite cadastrar novos produtos na lista.
- **Visualizar Lista:** Exibe todos os itens salvos de forma organizada (com validação para lista vazia).
- **Remover Itens:** Busca e deleta um item específico da lista através do nome.
- **Validação de Menu:** Sistema protegido contra opções inválidas digitadas pelo usuário.

## 🧠 Conceitos de Python Praticados
- Variáveis e tipos de dados.
- Estruturas condicionais (`if`, `elif`, `else`).
- Laços de repetição (`while` para o menu e `for` para percorrer a lista).
- Manipulação de listas em memória (`.append()`, `.remove()`, `len()` e o operador `in`).
